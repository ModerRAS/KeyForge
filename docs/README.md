## 🛠️ 技术栈

### 核心技术
- **框架**: .NET 9.0
- **架构**: 领域驱动设计 (DDD)
- **模式**: 清洁架构、Sense-Judge-Act
- **UI**: Windows Forms

### 开发工具
- **IDE**: Visual Studio 2022
- **版本控制**: Git
- **CI/CD**: GitHub Actions
- **包管理**: NuGet

### 测试框架
- **单元测试**: xUnit
- **集成测试**: xUnit
- **UAT测试**: 自定义框架
- **覆盖率**: Coverlet
